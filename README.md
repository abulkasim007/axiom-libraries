# Axiom Libraries

## Overview
Axiom Libraries is a Java/Gradle multi-module repository providing reusable abstractions and implementations for hosting, messaging, data and threading concerns.

## Modules (present)
- `host-abstractions`
- `configuration-abstractions`
- `serialization-abstractions`
- `threading-abstractions`
- `messaging-abstractions`
- `data-abstractions`
- `mongodb-implementations`
- `rdbms-implementations`
- `host-implementations`
- `rabbitmq-implementations`

## Publish
- Publish all modules to local Maven:
    - `.\gradlew.bat publishToMavenLocal`
